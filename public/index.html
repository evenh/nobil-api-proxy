Nobil API Proxy api

A simple front end for performing API calls to Nobil, using more sensible URI's. Created by Even Holthe in use for a Android school project at HiOA.

All calls require use of the X-Authorization header containing your API key. All parameters are required, unless they are marked as Optional.

Endpoint Parameters Method
/chargers/id/:charger_id

  • :charger_id The charger ID. Example: NOR_00171

/chargers/near/:lat/:lon?distance=500&limit=10

The nearest charger station

  • :lat Coordinate. Example: 59.91673
  • :lon Coordinate. Example: 10.74782
  • distance (Optional integer). Distance in meters. Default value: 500
  • limit (Optional integer). Default value: 10
 GET
/stats/:country

Gets total stats and counties for a given county

  • :country Country code. Valid codes are: NOR, SWE and DAN
 GET
/stats/:country/county/:county_id

Gets total stats for a given county

  • :country Country code. Valid codes are: NOR, SWE and DAN
  • :county_id A valid country id. Can be received by calling /api/stats/:country
 GET
/stats/:country/county/:county_id/municipalities

Gets total stats for all municipalities in a given county

  • :country Country code. Valid codes are: NOR, SWE and DAN
  • :county_id A valid country id. Can be received by calling /stats/:country
 GET
/stats/:country/municipalities/:municipality

Gets stats for a given municipality

  • :country Country code. Valid codes are: NOR, SWE and DAN
  • :municipality Municipality id. Can be received by calling /stats/:country/county/:county_id/municipalities
 GET
/stats/:country/municipalities/:municipality/detail

Gets detailed stats for a given municipality

  • :country Country code. Valid codes are: NOR, SWE and DAN
  • :municipality Municipality id. Can be received by calling /stats/:country/county/:county_id/municipalities
